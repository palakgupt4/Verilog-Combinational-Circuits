# Verilog-Combinational-Circuits

## Overview

This repository contains the implementation and verification of fundamental **combinational digital circuits** using **Verilog HDL**. Each module has been designed, simulated, and verified using **Xilinx Vivado** with dedicated testbenches. The repository serves as a structured learning portfolio covering the core combinational logic concepts taught in Digital Electronics and introductory VLSI Design.

---

## Objectives

* Design combinational digital circuits using Verilog HDL.
* Understand different Verilog modeling styles.
* Verify functionality using simulation testbenches.
* Analyze RTL schematics generated by Vivado.
* Build a strong foundation for FPGA and VLSI design.

## How to Run

1. Open the desired project in Xilinx Vivado.
2. Add the Verilog source file and its corresponding testbench.
3. Set the testbench as the top module.
4. Run **Behavioral Simulation**.
5. Verify the generated waveform.
6. Open **Elaborated Design** to view the RTL schematic.
